Blender

This page is about cool piece of software

Blender features:

  • Modelling Sculpting, retopology, modeling, curves. Blender`s modeling toolset is extensive.
  • Sculpting Digital sculpting tools provide the power and flexibility required in several stages of the digital production pipeline.
  • Rendering Create jaw-dropping renders thanks to Cycles, high-end production path tracer.
  • Animation & RiggingDesigned for animation, Blender is being used for award-winning shots and feature films.
  • VFX From camera and object motion tracking to masking and compositing, you can "fix it in post".
  • Grease Pencil Push the boundaries of Story Art and 2D concept design, right in Blender`s 3D viewport.
  • Simulation Featuring industry-standards libraries like Bullet and MantaFlow, Blender offers powerful simulation tools.
  • Video Editing The Vidoe Editor offers a range of basic yet very efficient tools.
  • Pipeline Integration with multiple pipeline tools is pivotal in many productions.
  • Scripting Featuring an extensive Python API, every tools is available for scripting and customisation.
  • Interface Because of its custom architecture, Blender`s UI, window layout and shortcuts can be fully customised.
